Sample Problem: Array Rotation

Rotate a given array to the right by n steps. After each step, the last element in the array is moved to the front of the array, and the rest of the elements are shifted right. Examples:

  • Given [1, 2, 3] and n = 1, you should return [3, 1, 2].
  • Given [1, 2, 3, 4, 5] and n = 3, you should return [3, 4, 5, 1, 2].
